BBC Micro:Bit

The BBC Micro:Bit is perfect for introducing children to coding and hardware. This pocket-sized computer supports coding languages like Python and Scratch, and comes with features like a built-in microphone. It also provides 5 free classes for kids aged 7-14 to help them get started in STEM education. After the free month, there is an option to extend learning. ideal for fostering creativity and innovation in Young learners.

Technical Description

The BBC Micro:Bit is an upgraded version of the original micro. With improved audio capabilities and it is compatible with ELECFREAKS kits, it retains the features that have been popular in educational settings. Its easy integration and expansion features make it ideal for innovative and educational activities.

Other features include:

MEMs Microphone with LED indicator

Touch-sensitive logo

Built-in sleep/off mode — making it possible for the board to be powered down with batteries connected.

Discrete regulator, which can supply up to 190mA of current to external accessories.

No Cables or Power Sources Included

The package does not include power sources. Users will need to provide their own.
